⚡ Dev Setup

leonard.work — 一键装机 & 快捷命令

🚀 一键装机

下载脚本后运行，自动检测平台：

curl -fsSL https://leonard.work/setup/setup.sh | bash

或指定平台：

curl -fsSL https://leonard.work/setup/setup.sh -o setup.sh && bash setup.sh wsl
curl -fsSL https://leonard.work/setup/setup.sh -o setup.sh && bash setup.sh ubuntu
curl -fsSL https://leonard.work/setup/setup.sh -o setup.sh && bash setup.sh mac
📥 下载 setup.sh 📦 Hub Skills

🐚 Shell 工具

Oh My Zsh + 插件

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions && git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting && git clone https://github.com/zsh-users/zsh-completions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-completions

Starship

curl -sS https://starship.rs/install.sh | sh

Zoxide

curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh

📦 SDK & 语言

NVM + Node.js

Rust

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y && source ~/.cargo/env

uv (Python)

curl -LsSf https://astral.sh/uv/install.sh | sh

🖥️ 桌面应用

WSL/Windows winget 批量安装

winget install Google.Chrome Tencent.WeChat WhatsApp.WhatsApp Anthropic.Claude Microsoft.VisualStudioCode wez.wezterm Docker.DockerDesktop Telegram.TelegramDesktop Discord.Discord Microsoft.PowerToys Obsidian.Obsidian --accept-package-agreements --accept-source-agreements

Mac brew cask 批量安装

brew install --cask google-chrome wechat whatsapp claude visual-studio-code wezterm docker telegram discord raycast alt-tab stats iina the-unarchiver obsidian

Ubuntu 常用应用

wget -qO /tmp/chrome.deb https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb && sudo dpkg -i /tmp/chrome.deb; sudo snap install telegram-desktop discord obsidian
