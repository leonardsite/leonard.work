0 前置判断

你需不需要上 EOP？

只有 第一次请 MDW 的雇主 才必须完成 EOP（Employer's Orientation Programme）。如果你之前名下请过女佣，系统通常不会再卡你。

💡 最稳的做法：先登录 FDW eService 试试能不能进下一步。如果系统没卡你，就不需要 EOP。
1 如需要

完成 EOP（仅限首次雇主）

至少在提交 Work Permit 申请前 2 个工作日 完成课程。

EOP 官方规则说明

报名上课直达链接：

GMC 报名 We Are Caring 报名
✦ 总入口

FDW eService 登录入口

下面所有操作，两边雇主都从这里登录。点 "For employer or sponsor → Log in with Singpass"。

打开 FDW eService 🔗
2 核心步骤
👤 新雇主（你）操作

发起 Transfer，请求现雇主同意

先确认两个前提：

  • ☐ Helper 已做 6ME（6-Monthly Medical Examination）
  • ☐ Work Permit 剩余 ＞ 30 天（不够的话，现雇主要先 extend）

在 eService 里的操作路径：

FDW eService › Apply for a Work Permit

在申请流程中，系统会让你：

Request for the current employer's consent for the transfer
📨 系统会自动发邮件给现雇主，通知对方来 approve。你不需要另外通知（但提前沟通好更稳）。
3 核心步骤
👤 现雇主操作

现雇主 Approve Transfer

  • 📩
    等 MOM 的邮件通知（没有邮件别乱点）
  • ✅
    登录 FDW eService → Approve transfer request
  • 💰
    Levy 持续缴到新 WP issued 那天；issued 后旧 WP 自动取消，levy 自动停
  • ✂️
    旧 Work Permit 卡 → 剪成两半丢掉（防滥用）
4 核心步骤
👤 新雇主（你）操作

正式提交申请，拿 IPA

现雇主同意后，你回到 eService 把申请提交完：

FDW eService › Apply for a Work Permit › 提交完成
⏱ 通常 1 周内出结果。批准后会给你 IPA（In-Principle Approval）+ Declaration Form（可在 eService 下载）。
5 准备材料
👤 新雇主（你）操作

拿到 IPA 后必须准备的材料

少任何一项，issuance 都会卡住！

  • 🔒
    Security Bond $5,000（马来西亚籍免）
    查看 Security Bond 详情 →
  • 🏥
    Medical Insurance + Personal Accident Insurance
    查看保险要求 →
  • 📝
    Declaration Form — 你和 helper 都签好名（从 eService 下载打印）
6 协调
👥 双方协商

约定 Transfer Date（交接日）

你和现雇主商定好具体哪天交接。

⚠️ MOM 提醒：6ME + 请求同意这两步，要在 WP 过期前 至少 30 天 完成。务必算好时间！
7 最终步骤
👤 新雇主（你）操作

Transfer Date 当天 — 正式 Issue 新 WP

交接日当天，你登录 eService：

FDW eService › Ask for a Work Permit to be issued

按顺序做 3 件事：

  • ①
    Issue 新 Work Permit
  • ②
    Upload Declaration Form（已签好的那份）
  • ③
    Print Temporary Work Permit (TWP)（给 helper 的临时证）
🎉 Issue 完成后：
· 旧 WP 自动取消，现雇主 levy 自动停
· 把 TWP 给 helper，她当天就能合法开工！

🚨 你必须盯死的 3 个"卡点"

1

现雇主不点同意 → 流程死锁

所以前期一定要先谈妥，确认对方会配合。

2

6ME 没做 / 没提交 → 申请被打回

你提交申请前就会被要求补，不要等到最后一刻。

3

拿到 IPA 拖着不 Issue → 会被罚

MOM 有明确截止规则（21 天 或 WP 到期日取更早），拖过会罚。
