# 汽车文明图片时间轴（儿童版）

面向儿童课堂的图片优先项目：大图卡片 + 条形图信息 + 自动跳转时间轴。

## 你现在会看到什么

- 大量车型图片（新增 50 条品牌畅销车型图卡）
- 时间轴进度条：按“有内容年份”吸附，不会拖到空年份
- 信息条形图（Bar）：
  - 销量 Sales
  - 产量 Output
  - 产地覆盖
  - 关税压力
  - 政策分
- 关税/补贴/油耗法规等政策面板，解释“为什么卖得多或少”
- 中文发音 + English voice + 引擎声

## 数据结构

- `data.js`：原有历史里程碑和车型
- `brand-models.js`：新增品牌畅销车型（含图片、销量、产地、政策分）
- `app.js`：交互逻辑与渲染
- `styles.css`：视觉样式（横屏投屏优化）

## 本地预览

```bash
cd /home/leonard/zhouruby.com/kids/auto-civilization
python3 -m http.server 8099
```

打开 `http://localhost:8099`

## 投屏建议（iPhone -> Apple TV 4K）

1. 手机与 Apple TV 同 Wi-Fi。
2. 打开页面后先横屏。
3. 控制中心 -> 屏幕镜像 -> 选择 Apple TV。
4. 课堂模式优先使用“知识点进度条”。

## 数据说明

- 销量与车型信息来自公开条目整理，适合教学展示。
- 部分历史产量/价格/政策影响分值为教学估算。
- 政策条用于解释趋势，不构成投资或政策解读建议。
